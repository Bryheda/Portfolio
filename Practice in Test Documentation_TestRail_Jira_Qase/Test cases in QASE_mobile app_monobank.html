  3. Клієнт має одну валідну гривневу картку Монобанк

Steps to reproduce

Step 1
Action

На головній сторінці натиснути на кнопку «Поповнити свою картку»
Input data
Expected result

Кнопка "Поповнити свою картку" натискається.
Відбувається перехід на сторінку "Поповнення своєї картки"
Step 2
Action

Обрати пункт «З іншої картки» в розділі «Інші способи»
Input data
Expected result

Відкривається вікно "Поповнити з іншої картки"
Step 3
Action

Ввести невалідний номер картки
Input data

4141 4141 4141 4141
Expected result

Невалідний номер картки викликає появу зверху червоної тимчасово спливаючої підказки «Неправильний номер картки».
Кнопка "Продовжити" зберігає сірий колір і не натискається
Step 4
Action

Ввести невалідний номер картки - відмінні від цифр знаки
Input data

"+"
"-"
"."
"/"
"*"
"#"
"пробіл"
Expected result

Натискання не цифр не призводить до їх надрукування.
Кнопка "Продовжити" зберігає сірий колір і не натискається
Step 5
Action

Натиснути стрілочку, перейти на буквенну клавіатуру і ввести невалідний номер картки - букви латиницею і кирилицею
Input data

"F"
"Л"
Expected result

Натискання букв латиницею і кирилицею не призводить до їх надрукування.
Кнопка "Продовжити" зберігає сірий колір і не натискається
